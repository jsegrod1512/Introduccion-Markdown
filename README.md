## Tabla de contenidos Markdown

| **Etiqueta**    | **Descripción**                  |
|-----------------|----------------------------------|
| `# Encabezado`  | Crea un encabezado de nivel 1.   |
| `**Negrita**`   | Aplica negrita al texto.         |
| `*Cursiva*`     | Aplica cursiva al texto.         |
| `[Enlace](url)` | Crea un enlace a una página web. |
